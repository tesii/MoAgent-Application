
# MoAgent Analytics Module

## 📌 Project Overview
The MoAgent Analytics module is responsible for analyzing agent transaction behavior, focusing on cash-in and cash-out activities. It identifies patterns in transaction timing and highlights behavioral trends.

---

## 🎯 Scope and Objectives
- Analyze transaction behavior by day and hour
- Identify patterns in cash-in vs cash-out activities
- Generate insights for decision-making
- Support visual reporting through dashboards

---

## 🛠️ Technologies Used
- PHP (Data processing)
- MySQL (Data storage)
- Flutter (Visualization)
- Flychart (Graph rendering)

---

## 📊 Output
- Transaction trend analysis
- Behavioral insights
- Visual dashboards for decision support

---

## 🚀 Purpose
To transform raw transaction data into meaningful insights that help understand agent behavior and improve operational decision-making.

# MoAgent Backend API

## 📌 Project Overview
The MoAgent Backend API is responsible for processing and delivering agent transaction data used in the MoAgent system. It supports analysis of cash-in and cash-out activities, enabling insights into agent transaction behavior over time.

---

## 🎯 Scope and Objectives
- Analyze cash-in and cash-out transactions by day and hour
- Provide structured data for frontend visualization
- Support reporting and decision-making through APIs
- Enable efficient querying of transaction patterns

---

## ⚙️ Technologies Used
- PHP
- MySQL
- XAMPP (Apache Server)
- REST-style JSON API

---

## 📂 Key Features
- Secure API endpoints for data access
- Role-based data filtering
- Location-based transaction queries
- JSON response format for frontend integration

---

## 🔐 Purpose
This backend serves as the core data engine for the MoAgent system, powering analytics and dashboard visualizations.
